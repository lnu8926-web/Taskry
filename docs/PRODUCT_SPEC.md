## 1. 제품 개요

### 1.1 제품명

**Taskry** - 칸반보드 기반 프로젝트 관리 도구

### 1.2 비전

"개인의 생산성에서 시작해, 팀의 협업으로 확장되는 프로젝트 관리 플랫폼"

### 1.3 타겟 사용자

| 단계               | 사용자             | 핵심 니즈                            |
| ------------------ | ------------------ | ------------------------------------ |
| **Phase 1 (현재)** | 개인 사용자 (본인) | 개인 프로젝트/태스크 관리, 일정 파악 |
| **Phase 2 (미래)** | 소규모 팀 (2-10명) | 팀 협업, 역할 분담, 진행 상황 공유   |
| **Phase 3 (확장)** | 중규모 팀/조직     | 멀티 워크스페이스, 권한 관리         |

### 1.4 핵심 가치

- ✅ **심플함**: 복잡하지 않은 직관적인 UI
- ✅ **유연함**: 칸반, 캘린더 등 다양한 뷰 제공
- ✅ **확장성**: 개인 → 팀으로 자연스러운 확장

## 3. 정보 구조 (IA)

### 3.1 사이트맵

```
Taskry
├── / (홈/대시보드)
│   └── 최근 프로젝트, 오늘 할 일, 통계
│
├── /project/[id] (프로젝트 상세)
│   ├── ?view=kanban (칸반보드) [기본]
│   ├── ?view=calendar (캘린더)
│   └── ?view=list (리스트) [선택]
│
├── /project/new (프로젝트 생성)
│
├── /calendar (전체 캘린더) [선택]
│
└── /settings (설정)
    ├── 프로필
    ├── 테마 (다크모드)
    └── 데이터 관리
```

### 3.2 URL 구조

| URL                             | 설명                       |
| ------------------------------- | -------------------------- |
| `/`                             | 대시보드                   |
| `/project/abc123`               | 프로젝트 상세 (기본: 칸반) |
| `/project/abc123?view=calendar` | 프로젝트 캘린더 뷰         |
| `/project/new`                  | 새 프로젝트 생성           |
| `/settings`                     | 설정 페이지                |

## 5. 데이터 모델

### 5.1 Phase 1 (개인용)

```typescript
// 프로젝트
interface Project {
  id: string;
  name: string;
  description?: string;
  color: string; // 프로젝트 색상
  status: "active" | "completed" | "archived";
  started_at?: string;
  ended_at?: string;
  created_at: string;
  updated_at: string;
}

// 태스크
interface Task {
  id: string;
  project_id: string;
  title: string;
  description?: string;
  status: "todo" | "inprogress" | "done";
  priority: "low" | "normal" | "high";
  started_at?: string;
  ended_at?: string;
  subtasks?: Subtask[];
  memo?: string;
  created_at: string;
  updated_at: string;
}

// 서브태스크
interface Subtask {
  id: string;
  title: string;
  completed: boolean;
}
```
